To build this first install node.js 
then install vite in command line with 
```npm install -D vite```
change directory to frontend from memolyzer with 
```cd apps/frontend```
run in command line
```npm install```
change director to backend with 
```cd ../backend```
run in command line 
```npm install```
then run with return to root directory
```cd .../../```
lastly run both frontend and backend with
```npm run dev```
and click on the locally generated link
